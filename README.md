# TubBucket — Your Streaming Everything

TubBucket curates the best free streaming platforms on the internet — movies, anime, manga, live TV & more.

## 🚀 Deployment on Vercel

This project is ready to be hosted on Vercel. 

### Option 1: Vercel CLI (Recommended)
If you have Vercel CLI installed:
1. Open your terminal in this directory.
2. Run `vercel`.
3. Follow the prompts.

### Option 2: GitHub Integration
1. Push this code to a new GitHub repository.
2. Go to [Vercel](https://vercel.com/new).
3. Connect your GitHub account and import this repository.
4. Vercel will automatically detect the Vite configuration and deploy it for you.

## 📱 Features
- **Fully Responsive**: Optimized for desktop, tablet, and mobile.
- **Local Favorites**: Bookmark your favorite sites locally.
- **Instant Search**: Find what you need in a snap.
- **Modern UI**: Sleek dark/light mode with aurora effects.
